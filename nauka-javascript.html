JavaScript

  • odpowiada za interakcję
  • pozwala na programowanie we frontendzie
  • potrafi wpłynąc na kazdą z warstw
  • rozszerza mozliwości zwykłych stron
  • aktualnie jest jednym z najpopularniejszych języków programowania na świecie

ECMAScript6+

  • ECMAScript jest ustandaryzowany przez organizację ECMA-skryptowy język programowania
  • JavaScript implementuje rzeczy ze specyfikacji ECMAScript
  • rozwój ECMAScript mocno przyspieszył przez szerokie wykorzystywanie JavaScriptu
  • ES6 (i nowsze) wprowadzają wiele udogodnień dla developerów
  • nowe feauture'y mogą nie być wspierane we wszystkich przeglądarkach (ale są na to sposoby)
