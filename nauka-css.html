CSS

Odpowiadają za prezentację, wpływa na wygląd elementów HTML na ekranie, wydrukach itp. Składnia CSS to reguły.

Budowa reguły

selektor {
właściwości: wartość;
}

Prawdziwe selektory:

  • .(kropka)-selektor klasy

    <znacznik class="name">treść</znacznik>

     .name{
      właściwość:wartość;
     }
  • #(hash)-selektor id

    <znacznik id="name">treść</znacznik>

     #name {
      właściwość:wartość;
      }
  • !important-nadpisuje wszystko inne, raczej nie stosujemy

Specyficzność w CSS

  1. !important
  2. style=""
  3. #name
  4. .name
  5. element

Jeśli specyficzność jest równa-pierwszeństwo ma selektor "blizej" elementu.

Najlepiej dodawać klasy do wszystkich elementów i unikać zagniezdzeń.

Nazewnictwo klas

  • nazywaj rzeczy zgodnie z ich funkcją na stronie
  • startuj z małej litery
  • nie zaczynaj od cyfry
  • jeśli musisz uzyć dwóch słów uzyj tzw. kebab-case
  • staraj się, zeby element miał jedną klasę (nie zwiększaj specyficzności bez potrzeby)
  • uzywaj pojedyńczych słów z języka angielskiego

Poznane tagi

  • <style>

    <style>
    p {
    color:red;
    }
    </style>
  • <link>

    <link href="style.css" rel="stylesheet" type="text/css">
  • <znacznik style="color;yellow;">treść</znacznik> Styl przy znaczniku z wykorzystaniem atrybutu style nazywamy stylem inline.
